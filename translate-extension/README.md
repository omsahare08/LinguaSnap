# LinguaSnap – Instant Text Translator Extension

## Installation Steps

1. Open Chrome and go to: `chrome://extensions/`
2. Toggle ON **"Developer mode"** (top right)
3. Click **"Load unpacked"**
4. Select the `translate-extension` folder
5. The extension is now installed! 
## How to Use

### Method 1 – Select & Translate (Easiest)
- Highlight any text on any webpage
- Click the green **"Translate"** button that appears
- A popup shows the translation instantly

### Method 2 – Right-click menu
- Select text → Right-click → **"Translate with LinguaSnap"**

### Method 3 – Extension Popup
- Click the LinguaSnap icon in your toolbar
- Type or paste text, choose language, hit **TRANSLATE**

## Features
-  Instant translation on text selection
-  45+ supported languages
-  Copy translation with one click
-  Text-to-speech (Listen) for translations
-  Swap source/target languages
-  Open in Google Translate for full context
-  Translation history (last 20)
-  Customizable settings
-  Auto language detection
-  Works on ALL websites
-  Free (uses MyMemory API, 1000 req/day)

## API Used
- **MyMemory** - Free translation API (https://mymemory.translated.net)
- No API key required
- 1000 requests/day on free tier

## Shortcut
- In the popup: `Ctrl+Enter` to translate
